# Boating in Gazebo

This repository regroups several plugins found in various projects in order to simulate surface vessels, especially with wings and rudders. Sources are from:

  - Waves, visual and induced hull forces: [Virtual Robot X](github.com/osrf/vrx) and [asv_wave_sim](https://github.com/srmainwaring/asv_wave_sim)
  - Lift and drag: modified from the [official plugin](github.com/gazebosim/gz-sim) to take either wind or water current
  - Anemometer: inspired from [asv_sim](https://github.com/srmainwaring/asv_sim)
